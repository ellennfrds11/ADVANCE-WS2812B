# ADVANCE-WS2812B
Repositori kontrol LED WS2812B tingkat lanjut. ✨ Fitur:  Animasi berbasis matematika &amp; Perlin Noise.  Optimasi Power Injection &amp; RAM.  Support 2D Matrix &amp; High Frame Rate.  Kompatibel dengan ESP32/Arduino/Teensy.  Solusi tepat untuk proyek LED yang halus, efisien, dan profesional.
